# 働き方改革を推進するための関係法律の整備に関する法律の一部の施行に伴う関係政令の整備及び経過措置に関する政令

平成三十一年政令第百五十五号

働き方改革を推進するための関係法律の整備に関する法律の一部の施行に伴う関係政令の整備及び経過措置に関する政令 抄
